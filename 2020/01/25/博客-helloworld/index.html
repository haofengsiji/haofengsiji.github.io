博客-helloworld

​ 今天花了大概三个小时，终于从有到无把自己的学习博客搭建起来了。很简单，但还是遇到了一些坑。总之记录一下自己学习的过程，知新。

介绍

​ 使用的是github和hexo。Github 大家应该都知道，是一个全球的最大的开发者社区，主要功能是版本控制，大家可以免费的托管自己的代码，只要是开源的。Hexo 则是一个博客网站开发框架，让我们能够用markdown的格式来写博客，hexo则帮助我们将博客以网站的形式展示出来。

正文

​ 1.创建github账号,以及版本控制器git

​ 2.下载开发软件

​ 3.设置Github博客主页

​ 4.hexo的使用

​ 5.hexo部署到Github

1.创建github账号,下载git.

​ github地址：https://github.com/只需要邮箱就可以。

​ git是版本控制器，提供类似linux的命令行，来进行版本的控制。小白也可以下载git desktop, git的图形化版本。git的基本的操作看官方的文档就可以，核心指令也就只有几个。

​ git:https://git-scm.com/download/

​ git desktop:https://desktop.github.com/

2.下载开发软件

Node.js

​ 检查是否安装成功：

​ Win+R->cmd

​ 打开windows的command，输入npm -v, 可以看到版本号即安装成功。

hexo

​ 如果类比node.js为python，那么hexo 就像numpy, matplotlib一样，是packgage。

​ 安装hexo, 使用command: $ npm install -g hexo-cli

​ 检查是否安装成功：$ hexo -v

3.设置github博客主页

创建一个repository

​ 进入你的主页，就会看到如下图的内容，点击New,创建repository.

​ repository的名字必须是你的 username.github.io，这样的你的博客网页才能够正常浏览。比如我的：

​ 进入的repository的设置，找到GitHub Pages子设置项，点击蓝色的链接，就可已选择自己的博客主页的风格主题。

​ 随便选一个，因为我们最终用的是hexo的主题。完成后在点击链接，就可以访问自己的github博客主页了。此时如果会html,javascript的话，就可以设计主页等等。但为了加速开发博客，我们将使用hexo博客框架。

4.Hexo的使用

hexo初始化部署

​ 新建一个文件夹，比如E:\study\program\blog。在blog文件夹内，来做博客的开发。

​ 进入到该文件夹的路径下，点击右键打开

​ 这样就会有一个跟linux一模一样的命令行。在命令行内，输入$ hexo init 来初始化工程，你就会得到跟我一样如上图的那些文件了。然后输入$ npm install安装必要的组件。

​ 接着输入$ hexo g生成public的静态网页项目，然后输入$hexo s运行本地服务器（这一步，通常用来debug，调试用的）

​ 在浏览器中输入localhost:4000，就可看到用hexo生成博客主页。主页还是挺好看的。

5.Hexo部署到Github

​ 这一步需要使用到hexo的deploy功能。首先需要设置deploy参数。打开blog/_config.yml, 在文件的最后加入如下参数

​ 类型是git, repository是你的项目的地址, 版本分支：master。保存后，在git bash中输入：

$ hexo g 生成public的静态网页工程

$ hexo s 本地浏览器调试预览

$ hexo d 部署

​ 这样别人通过访问 https://yourusername.github.io 就可以浏览你的博客了。

​ 创建一个新的博客命令：$ hexo new post "My New Post"
